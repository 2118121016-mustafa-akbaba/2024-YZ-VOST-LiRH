1- Eğer 3 dersi de alıyorsanız sadece öğrenci numaranızla klasör oluşturarak dosyalarınızı yükleyiniz.

OgrenciNo/TasarimRaporu.tex, TasarimRaporu.pdf

Örnek: 
OgrenciNo_181236007/TasarimRaporu.tex
